# HR Data Analysis

This project contains an HR data analysis notebook focused on data cleaning, exploratory analysis, and visualizations. The goal is to explore workforce patterns such as department distribution, salary levels, satisfaction, and employee turnover.

## What’s Included
- Exploratory Data Analysis with Pandas
- Visualizations using Matplotlib
- Insights on employee distribution, salary, satisfaction, and attrition

## Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Files
- HR.ipynb – Main analysis notebook
- hr_data.csv – Dataset
